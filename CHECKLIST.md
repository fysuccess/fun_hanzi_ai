# 项目部署检查清单

在部署项目到生产环境之前，请确保完成以下检查项：

## 环境配置

- [ ] 已创建 `.env.local` 文件（开发环境）
- [ ] 已在部署平台配置环境变量（生产环境）
- [ ] `MONGODB_URI` 已正确配置
- [ ] `DEEPSEEK_API_KEY` 或 `OPENAI_API_KEY` 已配置
- [ ] MongoDB Atlas 网络访问已配置（允许所有IP或指定IP）
- [ ] API密钥有足够额度

## 代码检查

- [ ] 运行 `npm run lint` 无错误
- [ ] 运行 `npm run build` 构建成功
- [ ] 所有TypeScript类型检查通过
- [ ] 测试用例全部通过 (`npm test`)

## 功能验证

- [ ] 汉字搜索功能正常
- [ ] 笔顺动画可以正常播放
- [ ] 动画控制按钮（播放/暂停/重播）工作正常
- [ ] 速度调整功能正常
- [ ] 拼音显示正确
- [ ] 语音朗读功能正常（浏览器支持）
- [ ] AI内容生成正常
- [ ] 描红练习模式可以正常使用
- [ ] 缓存机制工作正常（第二次查询更快）

## 兼容性测试

- [ ] Chrome浏览器测试通过
- [ ] Firefox浏览器测试通过
- [ ] Safari浏览器测试通过（如适用）
- [ ] 移动端浏览器测试通过
- [ ] 响应式布局在不同屏幕尺寸下正常

## 性能检查

- [ ] 页面加载时间 < 2秒
- [ ] 笔顺动画加载时间 < 1秒
- [ ] AI内容生成时间 < 3秒（首次）
- [ ] 缓存查询时间 < 1秒
- [ ] 无明显的性能瓶颈

## 安全检查

- [ ] 环境变量未提交到Git仓库
- [ ] `.env.local` 已在 `.gitignore` 中
- [ ] API密钥未暴露在客户端代码中
- [ ] 输入验证正常工作
- [ ] 错误信息不泄露敏感信息

## 数据库

- [ ] MongoDB连接正常
- [ ] 数据库索引已创建（可选，提升性能）
- [ ] 数据库备份策略已配置（如适用）

## 文档

- [ ] README.md 已更新
- [ ] 部署文档已准备
- [ ] 环境变量配置说明已提供
- [ ] API文档已更新（如适用）

## 部署后验证

- [ ] 生产环境URL可以正常访问
- [ ] HTTPS已配置（生产环境必需）
- [ ] 域名DNS解析正常（如适用）
- [ ] 错误监控已配置（如适用）
- [ ] 性能监控已配置（如适用）

## 可选优化

- [ ] 已配置CDN加速
- [ ] 已配置缓存策略
- [ ] 已添加网站图标（favicon）
- [ ] 已配置SEO元数据
- [ ] 已添加网站分析（如Google Analytics）

## 紧急联系

- [ ] 已记录关键服务联系方式
- [ ] 已准备回滚方案
- [ ] 已配置告警通知
